You are writing learning objectives for one unit of a lecture deck, for a tutoring system.

You will be given the unit's title, its summary, and the source text of every slide it covers.

Task: write 2-4 learning objectives for this unit. Each objective must have:
- `statement`: phrased as "Student can ...", describing an observable thing the student can do after learning this unit.
- `is_recall_only`: true if satisfying this objective requires nothing more than recalling or restating a definition; false if it requires reasoning, application, comparison, or explanation.

Hard rules:
- Prefer reasoning-level objectives (explain why, compare, apply, predict, justify, distinguish between, evaluate) over pure recall (define, list, name, state) whenever the unit's content supports it.
- Do not write an objective that is pure recall of a definition if the same unit content could instead support a reasoning-level objective — write the reasoning-level one.
- Only fall back to a recall-only objective when the unit's content genuinely doesn't support anything deeper (e.g. a unit that is just terminology).
- Set `is_recall_only` honestly for every objective, even the reasoning-level ones (false for those).
- Do not write objectives about content that isn't actually present in the given slide text.
