You are building an evidence card for one learning objective in a tutoring system.

You will be given the objective's statement, the exact source text of every slide in its unit, and (if this is not the first objective in the course) a numbered list of prior objectives already covered.

Task: produce expected ideas, known misconceptions, and prerequisites for this objective.

**expected_ideas** (2-5 of them): each is a short statement of one discrete piece of knowledge a student needs in order to satisfy the objective, paired with a `quote` — text copied EXACTLY, character-for-character, from the given slide text, that supports that idea. This is the single most important rule in this task:
- The quote must be copied verbatim from the slide text given to you — same words, same punctuation, same capitalization. Do NOT paraphrase, summarize, translate, or "clean up" the wording.
- Every idea must be atomic: one discrete fact or concept, not a compound sentence bundling several.
- If you cannot find a real, literal quote in the given text that supports an idea, do not include that idea at all. An idea with an invented or paraphrased quote is worse than no idea — it will be discarded automatically, and if it is discarded silently it becomes a bug your teammates cannot detect.
- Never invent an idea whose content is not actually present in the given slide text.

**known_misconceptions** (1-3 of them): plausible, specific ways a student might misunderstand this exact objective — not generic "students find this hard" statements. Each has:
- `code`: a short, stable identifier in snake_case (e.g. `shuffle_is_reduce`), unique within this objective.
- `text`: one sentence stating the misconception itself, as something a confused student would believe.

**prerequisite_indices**: if a numbered list of prior objectives is given, return the indices (integers) of any prior objectives a student must already understand before this one — content this objective builds directly on top of, not just "earlier in the course." Return an empty list if this objective is foundational or doesn't clearly depend on a specific prior one. If no prior-objectives list is given, always return an empty list.
